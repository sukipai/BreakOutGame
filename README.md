# Breakout (2D 打砖块游戏 & 轻量级渲染引擎)

这是一个基于 **C++23** 和 **OpenGL** 开发的经典 2D 打砖块游戏项目。该项目不仅实现了完整的游戏逻辑，还构建了一个模块化的轻量级 2D 图形渲染引擎，将底层渲染（Engine）与高层游戏逻辑（Game）解耦。

## ✨ 项目特性

### 1. 轻量级图形引擎 (Engine)
* **资源管理**：通过 `ResourceManager` 统一管理着色器、纹理和字体的加载与生命周期。
* **核心渲染器**：包含专门用于 2D 渲染的 `SpriteRenderer` 和用于高质量文本显示的 `TextRenderer`。
* **着色器系统**：支持顶点、片段和几何着色器的动态解析与编译。
* **粒子系统**：实现了基于对象池的 `ParticleSystem`，用于生成流畅的小球拖尾等视觉效果。
* **字体支持**：集成 `FreeType` 库，支持加载 `.ttf` 字体并生成纹理图集。
* **3D 扩展基础**：内置了 `Camera` 类 并通过 `Assimp` 库支持模型与网格加载（`Model`, `Mesh`），具备扩展至 3D 的潜力。
* **日志系统**：基于 `spdlog` 实现了分级的彩色日志输出（ENGINE/APP）。

### 2. 游戏功能 (Game)
* **状态机管理**：支持主菜单、关卡选择、游戏运行、暂停、游戏结束及胜利等多种状态。
* **物理碰撞**：实现了圆（BallObject）与 AABB 矩形（GameObject/Brick）之间的精确碰撞检测及反弹逻辑。
* **数据驱动关卡**：通过 `.lvl` 文件加载关卡布局，支持多种类型的砖块（普通、坚固等）。
* **实体系统**：基于 `GameObject` 基类构建了玩家挡板和小球实体。

## 📁 目录结构

* `Engine/`: 引擎核心代码，包括渲染、资源管理、日志和数学工具。
* `Game/`: 游戏特有的逻辑实现、关卡数据及资源文件。
* `Game/Resources/`: 包含着色器、纹理图片、关卡配置和字体文件。

## 🛠️ 技术栈与依赖

* **语言标准**：C++23
* **图形 API**：OpenGL 3.3 (Core Profile)
* **依赖库**：
    * **GLFW**：窗口创建与输入处理。
    * **GLAD**：OpenGL 函数加载。
    * **GLM**：数学运算库。
    * **FreeType**：字体渲染。
    * **Assimp**：模型加载。
    * **spdlog**：日志记录。
    * **stb_image**：纹理图片解析。

## 🚀 构建与运行

### 环境要求
* macOS (推荐，已配置 Homebrew 路径) 或其他支持 C++23 的环境。
* CMake (版本 >= 3.14)。

### 编译步骤
1.  **修改路径**：根据您的本地路径，在 `CMakeLists.txt` 中更新各第三方库（如 `glfw`, `glad`, `freeType` 等）的 `ROOT` 变量。
2.  **执行构建**：
    ```bash
    mkdir build && cd build
    cmake ..
    make
    ```
3.  **启动游戏**：
    ```bash
    ./breakout
    ```

## 🎮 操作说明

* **A / D**：向左或向右移动挡板。
* **Space**：发射小球。
* **ESC**：在游戏中暂停或返回菜单。
* **Enter**：菜单确认选择。
* **方向键**：菜单导航或关卡切换。
